Project : How Does Hardware Work?

This project focuses on designing and constructing basic logic gates using discrete transistors and fundamental electronic components. Working as part of a small team, we implemented and tested core digital logic functions such as AND, OR, and NOT gates. The project helped me understand how high-level digital logic is physically realized at the hardware level and strengthened my practical skills in circuit analysis, teamwork, and problem-solving.
